# <center>数据库学习</center>

## 1.建立一个新库
`create database db1;`

注意有冒号的
```mysql
mysql> create database db1;
Query OK, 1 row affected (0.00 sec)
```

## 2.查看现有库
`show databases;`有冒号，复数形式
```
mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| db1                |
| mysql              |
| performance_schema |
| sakila             |
| sys                |
| world              |
+--------------------+
7 rows in set (0.00 sec)
```
## 3.销毁一个库

`drop datebase db1;`再显示为
```
mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sakila             |
| sys                |
| world              |
+--------------------+
6 rows in set (0.00 sec)
```
